# 🛒 Full-Stack E-Commerce Platform (Next.js)

A complete, ready-to-use e-commerce platform built with Next.js, TypeScript, and Tailwind CSS.  
This project is designed for developers, startups, or anyone who wants to launch a modern online store without building everything from scratch.

It comes with a polished storefront, a powerful admin dashboard, secure authentication, and multiple payment integrations — all wrapped in a clean, premium UI.

---

## 🚀 What You Get

### 🛍 Modern Storefront
- Clean, premium UI with smooth animations and gradients  
- Fully responsive (mobile, tablet, desktop)  
- Easily customizable sections (hero, featured products, etc.)  
- Designed to look modern and feel fast  

---

### 🛡 Admin Dashboard
- Overview of users, revenue, and store activity  
- Manage products, users, and site settings  
- Track admin actions with logs (IP + timestamps)  
- Simple and powerful interface — no clutter  

---

### 🔐 Authentication & Security
- Secure authentication with NextAuth  
- Social login (Discord & Telegram)  
- Protected routes (server-side)  
- reCAPTCHA + 2FA support for extra security  

---

### 💳 Payments
- Stripe (credit/debit cards)  
- PayPal integration  
- Crypto payments via Hoodpay  

---

### 🌍 Global Features
- Multi-currency support  
- Multi-language support (English, Spanish, French)  
- Ready for international users  

---

## 🧑‍💻 Tech Stack

- Next.js 16 (App Router)  
- TypeScript  
- Tailwind CSS  
- Framer Motion  
- MongoDB (Mongoose)  
- Cloudinary (image storage)  
- NextAuth
